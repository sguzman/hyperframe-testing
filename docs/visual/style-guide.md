# Visual Style Guide: Iconographic Kinetic Documentary

This guide codifies the visual language inspired by the Ahoy "Iconic Arms" aesthetic, tailored for use with Hyperframes.

## 1. Aesthetic Philosophy: Propaganda Modernism
The goal is **Visual Authority**. Every frame should feel like a designed artifact from a museum or a historical archive.

- **Minimalism**: Sacrifice detail for structure. Use silhouettes instead of photos.
- **High Contrast**: Bold shapes against flat backgrounds.
- **Iconography**: Turn abstract concepts (Economy, Ideology, Technology) into geometric tokens.

## 2. Color System: Semantic Grammar
Color is not decorative; it is **semantic**. Use limited palettes (3-5 colors) where each color represents a narrative faction or state.

| Category | Typical Color | Narrative Meaning |
| :--- | :--- | :--- |
| **Power/Ideology** | `#E63946` (Red) | Origin, Revolution, Danger, Aggression |
| **Technical/Logic** | `#457B9D` (Blue) | Engineering, Precision, Systems, Bureaucracy |
| **Value/Wealth** | `#FFB703` (Gold) | Production, Mass Circulation, Economic Force |
| **Legacy/Finality** | `#1D3557` (Navy/Steel) | Closure, History, Monumental weight |
| **Neutral/Data** | `#F1FAEE` (Off-white) | Labels, Facts, Contrast |

## 3. Typography: Narrative Percussion
Text is a graphic mass that "hits" the viewer.

- **Primary Font**: Montserrat (Black 900) or Outfit (ExtraBold 900).
- **Secondary Font**: Space Grotesk (Light 300) for technical labels.
- **Styling**: All-caps, tracked-in (-0.05em), high contrast.
- **Function**: Text should appear exactly on a narrative beat, acting as punctuation.

## 4. Composition Rules
- **The Power Diagonal**: Use 45° or 15° diagonal fields to create dynamic tension.
- **The Isolated Icon**: Place a single, flat silhouette in the center of a radial gradient for maximum focus.
- **The Dossier Layout**: Use small technical labels (18-24px) with leader lines to identify parts of a larger whole.
